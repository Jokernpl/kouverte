# 🚀 DEPLOY SU REPLIT (GRATUITO)

## Procedura (5 minuti):

1. Vai su https://replit.com
2. Clicca "+ Create" 
3. Scegli "Import from GitHub"
4. Incolla questo URL:
   (Oppure carica i file manualmente)

5. Replit crea automaticamente:
   - Ambiente Node.js
   - URL pubblico permanente
   - Auto-restart se crasha

6. Nel terminale Replit, esegui:
   ```
   npm install
   node vox-server.js
   ```

7. Replit genera un URL tipo:
   ```
   https://Kouverte-Voice.broke.replit.dev
   ```

8. Usa quello nel bot Telegram!
